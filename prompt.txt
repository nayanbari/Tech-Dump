Please provide the code for these files."

"Step 2: Implement user authentication and secure token storage.

4.	Create a custom SQLAlchemy TypeDecorator that uses the cryptography library's Fernet symmetric encryption to automatically encrypt the user's Atlassian token before saving it to SQLite, and decrypt it when retrieving it.

Please write the code for the models, the TypeDecorator, and the auth routers."

1.	Define a TypedDict for the graph state tracking: message history, extracted Atlassian context, current JSON schema draft, preview data, and a boolean is_approved flag.
